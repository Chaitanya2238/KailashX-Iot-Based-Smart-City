## Recording Demo Video

### Using OBS Studio (Free)
1. Download [OBS Studio](https://obsproject.com/)
2. Add "Display Capture" source
3. Record your ESP32 system in action
4. Upload to YouTube or save as GIF

### Using ScreenToGif (Windows)
1. Download [ScreenToGif](https://www.screentogif.com/)
2. Record the serial monitor + LED activity
3. Save as GIF (< 10MB for GitHub)
4. Place in this folder

### Screenshot Tips
- Capture serial monitor showing detection
- Photo of LED states (Red ON, Green ON)
- Close-up of hardware setup
- Use good lighting for hardware photos
